# CODE-SNIPPETS-FORMATOR

> Text formatter to .CODE-SNIPPETS format; adds quotes `" "` to each line of text.
> Converts tabs `\t` and 4 spaces `"    "` into the literal `"\t"`.

---

## Features

- Automatic translation of tabs and spaces into `.CODE-SNIPPETS` format.
- Real-time formatting as you type.
- Copy formatted output to clipboard with a single click.
- Cross-platform support (Windows/macOS/Linux) via PySide6.
- Supports large text inputs with multi-line handling.
- Adjustable font size for input/output fields.
